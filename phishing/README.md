## Ethical Use & Legal Disclaimer

This home lab is designed for educational and professional development purposes, with a focus on cybersecurity, system administration, and network defense.

All testing activities conducted within this environment—including phishing simulations, vulnerability scanning, and security assessments—are performed exclusively in a controlled lab setting or on systems that I own or have explicit permission to test.

Phishing exercises are used strictly as a learning tool to better understand detection, user awareness, and defensive strategies. These simulations are not conducted against unauthorized individuals, organizations, or public systems.

I am committed to following all applicable laws, ethical guidelines, and responsible disclosure practices. This project is intended to reinforce secure and ethical behavior in cybersecurity, with an emphasis on protecting users, systems, and data.

No tools, techniques, or configurations demonstrated in this repository are intended for malicious use.
