## Software Stack

This home lab uses a mix of infrastructure, security, monitoring, and administrative tools to simulate a small enterprise environment for IT support, SOC, and cybersecurity practice.

### Core Infrastructure
- **pfSense** — Firewall, routing, NAT, VLAN segmentation, and access control
- **Cisco Layer 3 Switching** — VLAN management and internal network segmentation
- **Windows Server 2022** — Active Directory, DNS, DHCP, Group Policy, and domain services
- **Hyper-V** — Virtualization platform for hosting lab systems and test environments

### Security Monitoring
- **Suricata** — IDS/IPS for network traffic inspection and threat detection
- **Sysmon** — Endpoint telemetry for process, network, and system activity visibility
- **Splunk** — SIEM platform for centralized log collection, search, dashboards, and alerting

### Administrative and IT Workflow
- **Jira** — Ticketing and issue tracking for help desk, incident response, and workflow practice

### Offensive Security and Testing
- **Kali Linux** — Security testing distribution for offensive tooling and lab simulations
- **Nmap** — Network discovery and port scanning
- **Metasploit** — Exploitation and controlled attack simulation
- **GoPhish** — Phishing simulation and awareness testing

### Vulnerability Management
- **OpenVAS / Greenbone** — Vulnerability scanning and assessment of lab systems

### Analysis and Troubleshooting
- **Wireshark** — Packet capture and protocol analysis
- **VirusTotal** — File, hash, and URL reputation analysis
- **ANY.RUN** — Interactive malware analysis sandbox

### Scripting and Automation
- **PowerShell** — Windows administration, automation, and system management
- **Python** — Scripting, automation, parsing, and security tooling support

### Planned Use Cases
This software stack supports practice in:
- Network segmentation and firewall administration
- Active Directory and Windows administration
- SIEM monitoring and alert analysis
- IDS/IPS detection workflows
- Ticketing and help desk processes
- Vulnerability management
- Phishing simulation
- Malware analysis
- Basic incident response and documentation
