# Title: Printer and Scanner Lost Connectivity Due to Loose Ethernet Cable

## Environment
- Device: Printer / scanner
- Network Connection: Ethernet
- Related Change: Switch was removed from the upstairs setup

## Problem Description
The printer and scanner stopped working after changes were made to the upstairs network setup.

## Symptoms
- Printer was not working
- Scanner was not working

## Initial Hypothesis
A wire may have been unplugged when the switch was removed from the upstairs setup.

## Troubleshooting Steps Taken
1. Assessed the physical network connections
2. Checked whether a cable needed to be plugged back in
3. Verified the printer Ethernet connection
4. Secured the printer Ethernet cable into the modem
5. Tested printer/scanner functionality again

## Root Cause
The printer Ethernet cable was not securely connected to the modem.

## Resolution
Secured the printer Ethernet cable into the modem.

## Validation
- Printer functionality returned
- Scanner functionality returned

## Lessons Learned
- Physical connectivity should be checked first after making network hardware changes
- Simple cable issues can affect multiple device functions at once
