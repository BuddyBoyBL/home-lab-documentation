# Active Directory and Cable Management Update

## Summary
Made additional progress on both the logical and physical sides of the home lab by continuing work on Active Directory group policies, improving cable management, adding cable identification methods, and preparing for future printer connectivity on the test PC.

## Updates

### Active Directory Progress
Continued working on **Active Directory group policies** as part of the domain and client management setup.

**Focus area:**
- group policy configuration and refinement

**Why this matters:**
- supports centralized management
- helps simulate a more structured enterprise environment
- builds experience with administrative control over client systems

### Additional Cable Management Sleeves Applied
Added more cable management sleeves to improve organization around the room.

**Areas improved:**
- test PC area
- modem area
- admin table area

**Benefits:**
- cleaner cable routing
- better protection for exposed cables
- improved visual organization across the workspace

### Multi-Colored Cat6 Cables Added
Added a set of **multi-colored Cat6 cables** to improve visual differentiation at the Layer 3 switch.

**Purpose:**
- make connections easier to identify at a glance
- reduce confusion when tracing cables
- improve long-term physical organization

**Current note:**
- these are connected and helping with visual distinction
- the network is not yet fully segmented in its final intended state

### Cable Marker System Started
Started using **cable markers** to distinguish cables that are still the same color.

**Current labeling approach:**
- **red** for admin cable
- **blue** for modem cable

**Why this matters:**
- improves cable identification even when cable colors do not differ
- helps reduce mistakes during troubleshooting or reconfiguration
- supports a more deliberate physical management system

### USB to Ethernet Adapter Added for Test PC
Got a **USB to Ethernet adapter** for the test PC.

**Planned use:**
- connect the test printer to the test PC in the future

**Why this matters:**
- supports the test environment setup
- adds flexibility for connecting additional hardware
- helps prepare for future lab functionality without needing major hardware changes

## Why This Matters
This update improves the lab in multiple ways:

- Active Directory work strengthens the administrative and enterprise-style side of the environment
- cable sleeves improve physical organization and safety
- multi-colored Cat6 cables make switch connections easier to understand
- cable markers improve identification where cable colors still overlap
- the USB to Ethernet adapter supports future printer integration on the test system

Together, these changes make the lab easier to manage, easier to troubleshoot, and easier to expand.

## Current Position
The home lab is continuing to become more structured both physically and logically. The environment is still in progress, but cable organization, device planning, and administrative setup are all moving in a clearer direction.

## Follow-Up Items
- continue refining Active Directory group policies
- complete future segmentation work
- keep applying cable labels where needed
- connect the test printer to the test PC using the USB to Ethernet adapter
