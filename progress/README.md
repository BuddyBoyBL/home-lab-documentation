
# Progress Overview

This section documents the ongoing build process of my home lab, including setup progress, design changes, roadblocks, and next steps.

The lab is being built in stages based on available hardware, budget, and practical networking needs. Because of that, some parts of the environment are fully implemented, while others are still being tested, adjusted, or planned.

This progress log is meant to show:
- what has been completed
- what issues have come up
- what changes were made to the design
- what the next steps are

The goal is not just to show the final environment, but also to document the decision-making and troubleshooting process behind the build.

## Current Focus

- preparing monitoring and logging tools
- planning for future pfSense firewall deployment
- updating documentation as the lab evolves
