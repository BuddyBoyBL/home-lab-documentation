# Physical Setup and Storage Update

Completed several smaller physical setup improvements related to cable management, workspace organization, and temporary storage planning for the home lab.

**Current state:**

### Additional Cable Sleeves Added
Installed several cable sleeves in the room to improve cable management.

**Areas improved:**
- Places where cables are stepped over in the room
- Near the switch and hardware area for a cleaner setup

**Benefits:**
- Reduced visual clutter
- Improved cable protection
- Lowered the chance of cables being stepped on, caught, or shifted

### Temporary Cable Identification Added
Marked the cable running upstairs with a gold Sharpie as a temporary identifier.

**Current state:**
- Functional for short-term identification

**Future improvement:**
- Replace with proper cable labels or cable markers for a cleaner and more professional result

### Future Cable Management Need Identified
Confirmed that additional cable sleeves will still be useful as the setup expands.

**Reason:**
- More sleeves will improve organization and protection as more systems and cables are added

### Future Cable Color-Coding Goal Identified
I want to eventually move to **multi-colored Cat6 cables** to improve organization.

**Purpose:**
- Make cable roles easier to identify at a glance
- Improve physical organization
- Reduce confusion when tracing or changing connections
- Support a cleaner and more deliberate long-term setup

**Example future use:**
- one color for management
- one color for endpoint/device connections
- one color for uplinks or infrastructure
- one color for temporary testing connections

**Note:**
- This is a future organization improvement rather than an immediate need
- Current cables remain functional, but color-coding would improve clarity later

### Visual Deterrent Added

**Purpose:**
- Add a basic visible deterrent around the setup area

**Note:**
