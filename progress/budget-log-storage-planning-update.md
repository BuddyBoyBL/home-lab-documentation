# Title: Budget Log Storage Planning Update

## Summary
Made a storage planning decision to avoid keeping long-term logs on the admin laptop. Since the admin laptop is also a personal device, using it for long-term log retention would take up too much storage and make the setup harder to manage over time.

## Reason for Change
The admin laptop needs to preserve space for:
- personal use
- active lab work
- virtual machines
- current projects

Because of that, long-term log storage will be moved to separate hard drive storage rather than staying on the laptop SSD.

## Storage Approach
The plan is to use HDD storage for older logs and long-term archive purposes. This is a more budget-friendly option than relying on SSD storage for log retention.

## Hardware Purchased
Purchased two HDDs from eBay for about $20 each due to budget restrictions.

### Drive 1
- Western Digital WD10EZEX
- 1TB
- 7.2K RPM
- 6Gb/s
- 3.5" SATA HDD

### Drive 2
- Lenovo WD10EZEX-08M2NA0
- 1TB
- 3.5"
- 6Gb/s
- 7.2K RPM
- SATA Hard Drive
- FRU P/N: 45K0623

## Planned Integration
An internal HDD reader is already available and will be incorporated into the project. The current plan is to attach this setup through the docking station so that when the admin laptop is connected, necessary data can be backed up or offloaded without overwhelming the laptop’s internal storage.

## Long-Term Goal
This is the starting point for a larger storage plan.

Future goals include:
- expanding to 10TB or more
- moving into a proper case or enclosure that can hold multiple drives
- building redundancy for safer storage and backup
- creating a more dependable long-term storage platform

## Future Use Expansion
Long-term, the goal is not only to store older logs but also to eventually support storage and monitoring related to family devices. Being able to retain logs, set alerts, and monitor those systems would add peace of mind.

## Next Steps
- Check disk health on both purchased drives
- Integrate the drives into the archive workflow
- Begin using them for older log storage
- Continue separating active laptop storage from long-term archive storage
